# Portfolio — cookiecode.com.br

Personal portfolio site built with Next.js, deployed on Vercel.

## Quick start

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Updating content

Portfolio text (experience, skills, projects, etc.) lives in `content/`.  
See [docs/CONTENT.md](docs/CONTENT.md) and [AGENTS.md](AGENTS.md) for details.

## Pages

- `/` — main portfolio
- `/blog` — articles
- `/how-i-work` — working principles

## Deploy

Automatic deploy on push to `main` via Vercel.
